# portfolio-js-api

kakao developers에서 제공하는 검색 api를 활용한 책 검색하는 사이트

## 기억하고 싶은 코드

vite로 만든 바닐라 JS 프로젝트를 배포할 때,

```shell
npm run build
npm switch -C gh-pages
git add dist -f
git commit -m "deploy"
git subtree push --prefix dist origin gh-pages
```

이후, 저장소 'Settings > Pages > Build and deployment'에서 배포 브랜치를 설정해주면 된다.
